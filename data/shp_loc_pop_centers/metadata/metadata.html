Section 1: Overview

Originator:Minnesota Geospatial Information Office (MnGeo)

Title: City and Township Population Centers, Minnesota

Abstract: This layer contains cities, townships, reservations, and unorganized territories in the state of Minnesota, represented with point geometry for cartographic purposes. This dataset generally includes only populated places; there may be other named townships and territories which were not included. The population counts for these populated places have also been included, based on recent available US Census data.

Purpose: The dataset was developed largely for cartographic purposes, for the display of concentrations of populations. It can be symbolized with the available attributes to show the largest cities in the state or county (typically county seats), as well as very small populated cities, townships, unorganized territories, and tribal populations. The dataset can also be symbolized by population count where figures are available.

Time Period of Content Date: 03/01/2017

Currentness Reference: Data was created by referencing a 'CTU' (cities, townships, unorganized territories) database created and updated by MnGeo in 2009. The population column was added by MnGeo in 2017.

Section 2: Data Quality

Attribute Accuracy:

Logical Consistency: There should be only one point per place name per county. Some place names exist in more than one county, but in those cases represent different entities.

Completeness: This dataset generally includes only populated places; there may be other named townships and territories which were not included.

Horizontal Positional Accuracy: Source layer ctu: No known issues. Points were generally placed according to coordinates provided by MnGeo.

Vertical Positional Accuracy: N/A

Lineage: Originally developed by the Minnesota Department of Agriculture for cartographic purposes. Population column added March 2017.
